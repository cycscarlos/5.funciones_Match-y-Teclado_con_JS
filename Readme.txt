
Javascript Learning!

Seriado #5.                 

Contenido:

"Dibujar un circulo y hacer rotar una flecha a la derecha o izquierda, mostrando los grados recorridos de la circunferencia"

Temática:

1. Se creará una DIV y dentro de ella se colocará una imágen y una flecha
2. Al hacer click sobre el circulo se disparará una función
3. Dentro de esa función se validará la dirección de rotación de la flecha y se mostrarán los grados a los que se desplaza, bien sea a la derecha o a la izquierda
4. Se reutiliza el código usando funciones para actividades comunes
5. Se construyeron tres APPs, todas hacen lo mismo pero de diferentes maneras!
